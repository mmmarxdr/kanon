# Kanon

Kanon is an opinionated, AI-native project management platform. TypeScript monorepo with pnpm workspaces.

See AGENTS.md for full project details, tech stack, conventions, and available skills.

## Quick Reference

- **Setup:** `pnpm install && pnpm dev:start`
- **Test:** `pnpm test:all`
- **MCP tools:** Use `kanon_*` tools for project management (issues, boards, roadmap)
- **Skills:** Available in `.agent/skills/` -- load on-demand for project onboarding, issue management, and roadmap tracking
